# GitHub Pages сайт для бота MAX (ChatGPT 5.2)

## Что внутри
- `index.html` — главная страница
- `privacy.html` — политика конфиденциальности (шаблон)
- `styles.css` — стили
- `assets/avatar.png` — ваша картинка бота

## Что нужно заменить перед публикацией
1. В `index.html` замените ссылку:
   - `https://max.ru/your_bot_link` → на вашу ссылку в MAX
2. В блоке правил и контактов укажите:
   - имя/контакты владельца (email/Telegram)
3. При необходимости обновите тексты под свой проект.

## Как загрузить в GitHub
1. Откройте репозиторий
2. Нажмите `Add file` → `Upload files`
3. Перетащите все файлы из этой папки
4. Commit changes

## Как включить сайт (GitHub Pages)
1. `Settings` → `Pages`
2. `Source` → `Deploy from a branch`
3. `Branch` → `main`, `Folder` → `/ (root)`
4. `Save`

Ссылка будет вида:
`https://ВАШ-ЛОГИН.github.io/ИМЯ-РЕПОЗИТОРИЯ/`
